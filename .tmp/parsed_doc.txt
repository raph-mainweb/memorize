QR-Code System — Dynamische Zuweisung Das technische Herzstück des Medaillon-Modells: Medaillons werden in China mit fixen, eindeutigen Short-Codes vorproduziert. Die Verknüpfung zur Gedenkseite passiert erst beim Kauf — automatisch in der App. Kein Nachgravieren, kein Warten. Konzept: Short-Code + Redirect Auf dem Medaillon QR-Code zeigt auf: nachklang.ch/m/A7K3 (kurze, fixe URL) Beim Scan App liest A7K3 → sucht in DB → leitet weiter zu /gedenken/maria-baumann Wann zugewiesen Erst nach erfolgreichem Kauf — automatisch via Stripe Webhook Vor dem Kauf /m/A7K3 zeigt: «Dieser Code ist noch nicht aktiviert» Vorteile Sofortversand, keine ext. Produktion, volle Kontrolle über Lager Short-Code Format Format: 4 Zeichen — Grossbuchstaben + Zahlen, z.B. A7K3, B2X9, C4M1 Mögliche Kombinationen: 1.6 Millionen — reicht für sehr lange Zeit Auf Medaillon graviert: QR-Code + Text «nachklang.ch/m/A7K3» darunter Ablauf: Von Bestellung bis Grab # Schritt Was passiert 1 China-Batch bestellen 100 Medaillons mit Codes A0001–A0100 graviert (einmalig ~400 CHF) 2 Stock einlagern Codes in Supabase DB eintragen — status=available 3 Kunde kauft Stripe Zahlung 49 CHF erfolgreich 4 Auto-Zuweisung App nimmt nächsten freien Code (z.B.
